
## 3. 关键使用流程

## 3.1 新增一个新达人

```mermaid
flowchart LR
    A["发现达人链接"] --> B["新增达人"]
    B --> C["选择平台 + 粘贴链接"]
    C --> D["系统自动识别并回填资料"]
    D --> E["人工补充和修正"]
    E --> F["保存"]
    F --> G["进入达人详情页"]
    G --> H["上传主页截图/审美参考图"]
```

## 3.2 达人跟进流程

```mermaid
flowchart LR
    A["未联系"] --> B["已联系未回复"]
    B --> C["已回复沟通中"]
    C --> D["待报价"]
    D --> E["已报价待确认"]
    E --> F["已寄样"]
    F --> G["合作进行中"]
    G --> H["已发布内容"]
    H --> I["已完成"]
    C --> J["不合作"]
    D --> J
    E --> J
```

## 3.3 删除达人流程

```mermaid
flowchart LR
    A["点击删除"] --> B["填写删除原因"]
    B --> C["系统标记 is_deleted"]
    C --> D["记录删除时间"]
    D --> E["达人库默认隐藏"]
    E --> F["在已删除视图中红色+删除线展示"]
```

## 3.4 给领导导出流程

```mermaid
flowchart LR
    A["达人库筛选结果"] --> B["勾选达人"]
    B --> C["加入汇报名单"]
    C --> D["进入汇报导出页"]
    D --> E["调整顺序/删减"]
    E --> F["导出简介表格版 PDF / Excel"]
```
